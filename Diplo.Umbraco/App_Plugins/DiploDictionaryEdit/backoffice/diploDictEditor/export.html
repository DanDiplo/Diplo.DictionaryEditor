Dictionary Exporter
Export dictionary items to CSV

Overview

You can export the entire dictionary to a CSV file that you can then edit in spreadsheet editing software, such as Microsoft Excel or Google Sheets

You can then upload the edited CSV file to import it back into Umbraco using the Import Dictionary facility.

Download CSV File

Editing Help

When you download your CSV file it will have six columns - you should only update the last column called Translation. It is important you leave all other columns exactly as they are otherwise you won't be able to reimport it. Only use plain text and keep the text limit to a few hundred characters at most.

Note that you cannot add new values to the import list - it won't create new entries. It is only for updating existing values.
