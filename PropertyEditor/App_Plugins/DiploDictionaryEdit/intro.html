Umbraco Dictionary Editor

Overview

Here you will be able to edit existing dictionary items so you can translate them into all of the available languages.

Options

Edit Dictionary - allows you to edit dictionary items within Umbraco.

Export Dictionary - exports the dictionary to a CSV file for editing off-line.

Import Dictionary - import a CSV file you have edited off-line.

Note: This area allows you to edit existing dictionary items - you cannot create new items. If you wish to create new items you will need access to the Settings Section within Umbraco.
